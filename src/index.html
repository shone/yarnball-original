Woven Data
Yarnball defines three things:
Node
Link
Web
A node is a randomly generated number.
It can represent things, concepts, or anything.
For convenience, a node can be given a name. For example:
A link connects three nodes together.
It connects from the first node, via the second node, to the last node.
A collection of links is called a web.
A web can represent any kind of information, structured in any arbitrary way.
